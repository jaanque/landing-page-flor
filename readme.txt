ACTIVITAT 2: LANDING PAGE RESTAURANT - ESTILITZACIÓ AMB CSS
============================================================

CONTINGUT D'AQUESTA CARPETA:
- index.html: Estructura HTML base (personalitza el contingut)
- styles.css: Fitxer CSS buit (aquí faràs tota la feina)
- README.txt: Aquest fitxer

PASSOS A SEGUIR:

1. Tria el tipus de restaurant (pizzeria, sushi, hamburgueseria...)

2. Personalitza els textos de l'index.html:
   - Canvia [Nom Restaurant] pel nom del teu restaurant
   - Canvia [Slogan] pel teu slogan
   - Omple tots els textos entre [corxets]
   - NO modifiquis l'estructura HTML

3. Tria 2 fonts de Google Fonts i afegeix-les al <head>

4. Crea la teva paleta de colors.

5. Defineix les variables CSS al styles.css

6. Estilitza tots els elements seguint l'enunciat

7. Valida el teu codi (W3C Validator)

8. Crea el document reflexiu amb justificacions

9. Entrega la carpeta comprimida
